Para probar su programa utilicen el Makefile. Solo ejecuten 'make run-small' para probar el puzzle de 2x2.

Si la ejecución da algún problema y menciona que spark-submit no existe, modifiquen el Makefile para indicar la ubicación de este. Cambien la parte de 'spark-submit' a '~/spark-1.3.0/bin/spark-submit', o a la ruta correspondiente según su instalación.
